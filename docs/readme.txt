Название проект: FoodManager
Цель: Автоматизация процесса заказа, контроля продуктов на складе и кухне, и выдача отчетов по закупке продуктов для кухни.
Архитектура: Микросервисы
Описание: Присутствуют веб портал с разделением доступа по ролям: Повар, Менеджер по закупкам. Менеджеру разрешено генерировать отчеты по необходимых продуктов для работы кухни, продуктов с истекающим сроком годности и другие, выдача отчетов с анализом мест закупки и цен на продукты. Повару разрешено изымать продукты со склада на кухню, а также обеспечивать приемку продуктов на склад. Приложение позволяет парсить актуальные данные с сайтов сетевых магазинов и помещать в БД.
